Zapret

Мощный обход DPI для Windows и Linux

Что это: Обходит DPI (Deep Packet Inspection) на уровне системы. Не шифрует весь трафик, как VPN, а только подменяет пакеты, что обеспечивает максимальную скорость для YouTube и Discord.

📝 Пошаговая инструкция:

  1. Скачайте последнюю сборку с GitHub (Flowseal) или оригинального репозитория.
  2. Распакуйте архив в любую папку (желательно, чтобы в пути не было кириллицы).
  3. Запустите файл install_service.bat или general.bat от имени Администратора.
  4. В появившемся консольном окне выберите стратегию (для начала попробуйте вариант 1 или 3).
  5. Важно: Не закрывайте чёрное окно консоли, пока пользуетесь интернетом. Для проверки откройте YouTube.
⚡

ByeDPI

Лёгкий и быстрый обход блокировок (включая Android)

Что это: Более простая и легковесная альтернатива Zapret. Отлично работает на Windows, Linux и, что важно, на Android без root-прав.

📝 Пошаговая инструкция:

  1. Для Windows/Linux: Скачайте релиз с GitHub. Запустите исполняемый файл. Программа создаст локальный SOCKS5 прокси на адресе 127.0.0.1:1080.
  2. Пропишите этот прокси в настройках вашего браузера или системы.
  3. Для Android: Установите приложение ByeDPI Android. Нажмите большую кнопку Start. Весь трафик телефона начнёт обходить блокировки автоматически.
✈️

tgwsproxy

Специализированный прокси только для Telegram

Что это: Узкоспециализированный инструмент, который позволяет подключаться к Telegram через WebSocket, обходя блокировки провайдера, когда обычные MTProto прокси не работают.

📝 Пошаговая инструкция:

  1. Скачайте сборку для вашей ОС с GitHub (Desktop) или для Android.
  2. Запустите программу. Она автоматически сгенерирует специальную ссылку для подключения.
  3. Нажмите на ссылку, чтобы открыть Telegram и добавить прокси в один клик.
  4. Программа свернётся в трей (на ПК) или будет работать в фоне (на Android), обеспечивая стабильную связь.
