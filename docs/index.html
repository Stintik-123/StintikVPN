Как настроить за 3 шага

1
📋

Скопируйте ссылку

Выберите подписку выше и нажмите кнопку «Копировать». Ссылка сохранится в буфер обмена.

2
📲

Вставьте в клиент

Откройте VPN-клиент, нажмите «Добавить подписку» или «Импорт из буфера обмена».

3
🚀

Подключитесь

Нажмите «Пинг» для проверки серверов, выберите живой и нажмите «Подключить».
