Теория

🏴‍☠️ Чёрные и белые списки

🏴 Чёрные списки

Для обычного интернета (домашний Wi-Fi, кабель, 4G). Блокируют только запрещённые ресурсы.

🏳️ Белые списки

Для жёстких блокировок (мобильный интернет с «белыми списками» РКН). Пропускают только разрешённые ресурсы.

📡 Подробнее о белых списках: SNI и CIDR

White SNI: Фильтрует трафик по именам сайтов (например: youtube.com).
White CIDR: Фильтрует трафик по IP-диапазонам (например: 173.194.0.0/16).

💡 Если не работает SNI — используйте CIDR. Если не работает CIDR — используйте SNI.

Zapret

Мощный обход DPI для Windows и Linux

Что это: Обходит DPI на уровне системы. Не шифрует весь трафик, а подменяет пакеты, что обеспечивает максимальную скорость для YouTube и Discord.

📝 Пошаговая инструкция:

  1. Скачайте сборку с GitHub (Flowseal) или оригинального репозитория.
  2. Распакуйте архив (желательно без кириллицы в пути).
  3. Запустите install_service.bat или general.bat от имени Администратора.
  4. В консоли выберите стратегию (попробуйте 1 или 3).
  5. Важно: Не закрывайте окно консоли. Для проверки откройте YouTube.
⚡

ByeDPI

Лёгкий обход блокировок (включая Android)

Что это: Более простая альтернатива Zapret. Работает на Windows, Linux и Android без root-прав.

📝 Пошаговая инструкция:

  1. Windows/Linux: Скачайте с GitHub. Запустите — создаст SOCKS5 на 127.0.0.1:1080.
  2. Пропишите этот прокси в настройках браузера.
  3. Android: Установите ByeDPI Android. Нажмите Start.
✈️

tgwsproxy

Специализированный прокси только для Telegram

Что это: Позволяет подключаться к Telegram через WebSocket, обходя блокировки провайдера.

📝 Пошаговая инструкция:

  1. Скачайте сборку с GitHub (Desktop) или для Android.
  2. Запустите — программа сгенерирует ссылку для подключения.
  3. Нажмите на ссылку, чтобы добавить прокси в Telegram.
  4. Программа свернётся в трей (ПК) или будет работать в фоне (Android).
