📖 Инструкции

Обход блокировок (DPI)

️

Zapret

Мощный инструмент для Windows/Linux

Что это: Обходит DPI на уровне системы. Не шифрует весь трафик, а подменяет пакеты, что обеспечивает максимальную скорость для YouTube и Discord.

Инструкция:

  1. Скачайте сборку с GitHub (Flowseal) или оригинала.
  2. Распакуйте архив (в пути не должно быть кириллицы).
  3. Запустите install_service.bat или general.bat от имени Администратора.
  4. В консоли выберите стратегию (попробуйте 1 или 3).
  5. Важно: Не закрывайте окно консоли. Для проверки откройте YouTube.
⚡

ByeDPI

Лёгкий обход (Windows/Android)

Что это: Более простая альтернатива Zapret. Отлично работает на Windows, Linux и Android без root-прав.

Инструкция:

  1. Windows/Linux: Скачайте с GitHub. Запустите файл — создаст SOCKS5 на 127.0.0.1:1080.
  2. Пропишите этот прокси в настройках браузера.
  3. Android: Установите ByeDPI Android. Нажмите Start.
✈️

tgwsproxy

Прокси только для Telegram

Что это: Позволяет подключаться к Telegram через WebSocket, обходя блокировки провайдера, когда обычные MTProto прокси не работают.

Инструкция:

  1. Скачайте сборку с GitHub (Desktop) или Android.
  2. Запустите программу — она сгенерирует ссылку для подключения.
  3. Нажмите на ссылку, чтобы добавить прокси в Telegram.
  4. Программа свернётся в трей (ПК) или будет работать в фоне (Android).
