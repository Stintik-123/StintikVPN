## 🏴 Black & White Lists – When to Use Them

- **🏴 Black Lists** — for normal internet access (home Wi-Fi, cable internet, regular mobile data).
- **🏳️ White Lists** — for heavily restricted networks and mobile operators using strict filtering.

---

## 📡 More About White Lists: SNI & CIDR

- **White SNI** — filters traffic using domain names (example: youtube.com).
- **White CIDR** — filters traffic using IP ranges (example: 173.194.0.0/16).

**Simple explanation:**  
If SNI does not work — try CIDR.  
If CIDR does not work — try SNI.

---

## 🛠️ Protocols – Which One Should You Use?

- **VLESS** — best speed and traffic masking, but recently less stable because of active blocking.
- **Trojan** — excellent camouflage and reliability.
- **VMess** — one of the most stable protocols overall.
- **Shadowsocks** — maximum speed (recommended for online gaming).

## ❓ FAQ

### 1. The subscription does not work or is very slow.
Refresh the subscription inside your VPN client. Try another server or protocol. Restarting the client or changing Wi-Fi/mobile network may also help.

### 2. How often are configs updated?
Configs are updated daily by their original creators. README links always point to the latest versions.

### 3. When should I use white lists?
Only when your mobile operator enables strict filtering and regular internet access stops working correctly.

### 4. Which VPN client is best for beginners?
**Hiddify** is the easiest and most user-friendly option for Windows and Android. On iOS — **Streisand**.

### 5. What if Telegram proxy stops working?
Telegram proxies die frequently. Simply try another one from the list.

### 6. Can I use one subscription on multiple devices?
Usually yes, depending on the specific server.

### 7. Is it safe?
Configs are collected from public sources. Avoid sending highly sensitive information through public VPN nodes.
