# Contributing to StintikVPN

Thanks for your interest in contributing! 🎉

## How to Help

### Report Dead Links
If you find a broken subscription or proxy, please:
1. Open an issue with the title "Dead Link: [name]"
2. Include the broken link and when you found it
3. Suggest a replacement if possible

### Suggest New Configs
You can suggest working VPN configs by:
1. Opening an issue with "New Config: [name]"
2. Providing the subscription link
3. Including a brief description

### Improve Documentation
- Fix typos in README files
- Improve translations
- Add missing sections

## Guidelines

- Keep configs from **public sources only**
- Test links before suggesting
- Avoid promoting specific services
- Be respectful and constructive
