❓ FAQ

Подписка не работает

+

Обновите, выберите другой сервер или измените протокол

Какой клиент выбрать?

+

Hiddify - самый удобный. iOS - Streisand

Чёрные или белые списки?

+

Чёрные - для обычного интернета. Белые - только если оператор включил блокировки

SNI vs CIDR?

+

SNI - фильтр по сайтам. CIDR - по IP. Если не работает одно, попробуйте другое

Безопасно ли?

+

Из открытых источников. Не передавайте банковские данные
