📚 Гайды

🛡️ Zapret - Обход DPI на уровне системы

+

Что это? Самый мощный способ обхода блокировок. Работает на уровне операционной системы.

  1. Скачайте Zapret или оригинал
  2. Распакуйте архив в любую папку
  3. Запустите zapret-windows-x64.exe или run.bat
  4. Выберите стратегию (обычно первая)
  5. Оставьте окно открытым — это важно!

⚡ ByeDPI - Лёгкая альтернатива

+

Что это? Более простая, но менее мощная альтернатива Zapret. Хорош для новичков.

Windows/Linux:

  1. Скачайте ByeDPI
  2. Запустите исполняемый файл
  3. Прропишите в браузер прокси: 127.0.0.1:4242

Android:

  1. Установите APK из релизов
  2. Откройте приложение
  3. Нажмите кнопку Start

✈️ tgwsproxy - Прокси только для Telegram

+

Что это? Если нужен прокси только для Telegram через WebSocket.

  1. Скачайте tgwsproxy
  2. Запустите программу
  3. Кликните на ссылку в окне консоли
  4. Прокси добавится в Telegram автоматически!

❓ FAQ

Как пользоваться подписками?

−

1️⃣ Копируешь ссылку подписки 2️⃣ Открываешь VPN клиент (Hiddify, v2rayN, NekoBox и т.д.) 3️⃣ Вставляешь ссылку в поле «Добавить подписку» 4️⃣ Обновляешь и выбираешь сервер 5️⃣ Нажимаешь «Подключить» — готово!

Подписка не работает?

+

🔄 Обновите подписку в клиенте 🔀 Выберите другой сервер из списка ⚙️ Измените протокол (VLESS → VMess или Trojan) 📱 Попробуйте другой клиент 🔗 Проверьте что ссылка скопирована полностью

Чёрные или белые списки?

+

🏴 Чёрные списки — для обычного использования. Блокируют только запрещённые сайты. 🏳️ Белые списки — если оператор включил глубокие блокировки. Роутируют ТОЛЬКО разрешённые сайты.

Какой клиент лучше?

+

🏆 Новичкам: Hiddify (самый удобный) 📱 Android: NekoBox или Incy 🍎 iOS: Streisand 💻 Linux: Hiddify или NekoRay 🪟 Windows: v2rayN (больше настроек)

Безопасно ли использовать?

+

✅ Все конфиги из открытых источников и публичных репозиториев ⚠️ НЕ передавайте банковские пароли через публичные серверы 🔐 Используйте надёжные пароли для собственных аккаунтов 📍 Помните что оператор может видеть факт использования VPN

Как часто обновляются конфиги?

+

⏱️ Подписки обновляются автоматически каждые 24-48 часов 🚨 Если подписка «умирает» — просто выберите другую из списка 📢 Следите за Telegram каналом @StintikVPN для актуальных новостей
